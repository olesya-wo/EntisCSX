Имя функцииАргументыОписание
AddSelect flag, "str" Добавляет пункт выбора для диалога
BlackOut fHide, wait Исчезновение в темноту
BustupClear id Скрывает спрайт с указанным id. Если аргументов нет, то все спрайты.
BustupLeave accel, time, fFade, my, mx, id Скрывает спрайт с указанным id в направлении mx my за всемя time с ускорением accel
BustupMove pos, id Перемещает спрайт id в позицию pos
setCgInfo "buFile", "cgFile", flag, info Устанавливает псевдоним для фонов. Эти псевдонимы потом используются в SetCg
ChkFlagOn i Проверяет установленность флага i. После стоит число для смещени
ChkSelect i Метка при выборе в диалоге. После стоит число для смещения
Down accel, time, mv, id Смещение спрайта id по вертикали за время time
EyeCatchEnter fSoundKeep, type Горизонтальные чёрные полосы сверху и снизу экрана для смены главы
Flush "cgFile", time, color Вспышка цвета
Font "face", size Меняет размер и гарнитуру шрифта для сообщений
Hide fWait Скрывает окно сообщений.
Hitret num Ждёт клика игрока.
IsGameClear Проверяет, установлен ли флаг пройденности. Далее указано, сколько байт пропустить, если флаг не установлен
Jump mv, id Вертикальная встряска спрайта id
Mess "str" Выводит строку сообщения к имеющимся.
OnFlag i Устанавливает флаг i
OnGameClear Устанавливает глобальный флаг пройденности игры
Quake time, count, fWhole, h, w Дрожание камеры.
ScPauseBgm Приостанавливает фоновую музыку
ScPlayBgm fNonFade, "file" Начинает играть фоновую музыку
ScPlayEnvSe fFade, "file" Начинает играть зацикленный звуковой эффект
ScPlayMovie "file" Играет видеоролик
ScPlaySe fWait, "file" Проигрывает звуковой эффект единожды.
ScPlayVoice "file" Проигрывает голос
ScRestartBgm Перезапускает фоновую музыку
ScStopBgm fNonFade Останавливает фоновую музыку
ScStopEnvSe fFade, "file" Останавливает звуковой эффект
ScStopSe Останавливает звуковой эффект
ScWait fKeyDisable, time Пауза в игре
ScWaitVoice Ждёт, пока не закончится проигрывание голоса
Scroll fWait, accel, time, y, x Прокручивает фон сцены в позицию x y за время time
SetBustup priority, pos, "str" Выводит спрайт str в позиции pos c priority по z
SetCg "file", y, x Задаёт фоновое изображение file со смещением x y
SetCgRGB r, g, b Задаёт цвет фона вместо изображения
Shake count, mh, id Горизонтальная тряска спрайта id
StartSelect Выводит диалог для выбора
Talk "voice","name" Начинает диалог от имени name c озвучкой voice
Tone type Эффект. MONOCHROME, NEGATIVE и т.д.
Transition time, "file" Переход сцены за время time с эффектом file
Update fFlush Перерисовывает экран
WaitScroll Ждёт, пока закончится функция Scroll
WhiteOut fHide, wait Исчезновение в белизну
WindowView on Переключает режим(цвет) окна сообщений
Zoom fWait, accel, time, h, w, y, x Трансформация фона в точку x y с размерами области w h за время time
